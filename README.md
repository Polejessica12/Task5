# Task 5: Train-Test Split & Evaluation Metrics

## Objective
To train a machine learning model and evaluate its performance using standard metrics.

## Dataset
Heart Disease Dataset

## Steps Performed
- Loaded dataset
- Split data into training and testing sets
- Trained Logistic Regression model
- Predicted test data
- Evaluated model using accuracy, precision, recall
- Generated confusion matrix

## Tools Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## Outcome
This task helped understand how to evaluate machine learning models and interpret performance metrics.
